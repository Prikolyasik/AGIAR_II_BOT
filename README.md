# 🤖 IVAN INTELLEKTOVICH — Telegram Bot with AI

Telegram-бот с интегрированным искусственным интеллектом для аудита бизнеса.

## 📋 Описание

**IVAN INTELLEKTOVICH** — это высококвалифицированный эксперт по автоматизации бизнес-процессов и внедрению нейросетевых решений. Бот проводит глубокий аудит бизнеса пользователя, выявляет «узкие места» и предлагает стратегию автоматизации.

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка переменных окружения

Файл `.env` уже создан и содержит необходимые ключи:

```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
GROQ_API_KEY=your_groq_api_key
GROQ_MODEL=llama-3.3-70b-versatile
```

### 3. Запуск бота

```bash
# Для разработки (с авто-перезагрузкой)
npm run dev

# Для продакшена
npm start
```

## 🎯 Возможности бота

### Шаг 1: Сбор информации (Интервью)
Бот задаёт вопросы по одному или небольшими логическими блоками (не более 3 вопросов за раз):
- Ниша бизнеса и масштаб
- Текущие процессы
- Главная «боль»
- Используемые инструменты

### Шаг 2: Анализ и отчет
После сбора данных бот выдаёт структурированный отчет:
- Точки автоматизации
- Экономический эффект (снижение издержек, рост конверсии)
- Устранение человеческого фактора
- Прогноз по времени внедрения
- Impact-прогноз

### Шаг 3: Закрытие на целевое действие
Бот предлагает связаться с командой разработчиков для детальной настройки и интеграции.

## 🔧 Технологии

- **Node.js** — платформа выполнения
- **node-telegram-bot-api** — библиотека для Telegram Bot API
- **groq-sdk** — SDK для работы с Groq API (Llama 3.3 70B)
- **dotenv** — управление переменными окружения
